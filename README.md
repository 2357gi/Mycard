# Mycard
名刺を作ろう！
ARを組み合わせた個性的な名刺作りたい

# 名刺デザイン

## 名刺2次元デザイン
ARマーカーで6割ぐらい侵食されるので、それを踏まえたデザインが必要。


## 名刺3次元デザイン
土台、床が名刺として決まっているのでその名刺に書いてある文字の色が変わったり浮き出たり、
そういったものを作れたら楽しそう
